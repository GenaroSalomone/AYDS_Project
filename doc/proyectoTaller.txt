Sistema de preguntas y respuestas.
Software- Tecnología.

-1) Gestión de usuario, logearse con un gmail y contraseña.
-2) Selección de dificultad de las preguntas (Principiante - Intermedio - Experto).
-3) Diferentes tipos de preguntas (Choice - Autocompletado).
-4) Después de una trivia, proporcionar información sobre si la respuesta fue 
    correcta (o no), además, mostrar la respuesta correcta.
-5) Tiempo para preguntas: cada pregunta tendrá un tiempo límite.
-6) Las preguntas a responder serán diez, y éstas se seleccionan de un banco de x cantidad de preguntas.
-7) Si se elige dificultad principiante, se permitirá un sistema de ayudas/ pistas con información 
    adicional de la pregunta (incluso saltear una pregunta).
-8) Ranking de usuarios. Se podrá visualizar el ranking de puntos de cada usuario, donde se ira
    actualizando cada vez que se termina una trivia y se cargan los datos.
