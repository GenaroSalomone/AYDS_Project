#update_ranking(user, ranking, difficulty, score) ⇒ Integer

Updates or creates the ranking for a user based on the given score and difficulty.

This method checks if a ranking exists for the given user and difficulty level. If a ranking doesn’t exist or the new score is higher than the existing ranking score, it creates a new ranking or updates the existing ranking with the higher score. The method ensures that the user’s highest score is always saved.

Parameters:

  • user (User) —

    the user for whom the ranking needs to be updated

  • ranking (Ranking, nil) —

    the existing ranking record, or nil if it doesn’t exist

  • difficulty (Difficulty) —

    the difficulty level of the trivia

  • score (Integer) —

    the new score to potentially update the ranking with

Returns:

  • (Integer) —

    the highest score achieved by the user for the given difficulty level

def update_ranking(user, ranking, difficulty, score)
  if ranking.nil? || score > ranking.score
    ranking = Ranking.find_or_initialize_by(user_id: user.id, difficulty_id: difficulty.id)
    ranking.score = [score, ranking.score].max # Asegúrate de que siempre se guarde la puntuación más alta
    ranking.save
  end
  ranking.score # Devuelve la puntuación más alta, ya sea la nueva o la existente
end
